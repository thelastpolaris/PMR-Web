Welcome to PMR's Beta!

  On this webpage you can try person recognition in video and image files. The file processing is done asynchronously meaning that you can close the page and your video/image will still be processed (if no errors with the input will be encountered). Only 1 minute of the video due to limited hardware and possible multiple requests (anyway, it is a demo).
  Once processing is done, you can download the JSON file containing information about all the face bounding boxes and recognized persons per each frame and video/image file with drawn bounding boxes and persons names.
  List of all persons in the recognition dataset is here

Add new video task

Only .mp4 files. Please, choose small-sized files due to the slow internet connection of the server. Only 1 minute of the video will be processed.

Add new YouTube task

Provide the link to the video. Only 1 minute of the video with maximal resolution 480p will be processed.

Add new image task

.png, .jpg and .jpeg formats are accepted. Only one image is allowed.
